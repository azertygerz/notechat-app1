❓ Aide — Comment utiliser NoteChat

Tout ce dont tu as besoin pour bien démarrer

🔑Connexion & Inscription
1
Sur la page d'accueil, entre ton pseudo (ton nom d'utilisateur) et un code à 5 chiffres.
2
Si c'est ta première fois, ton compte est créé automatiquement. Si ton pseudo existe déjà, le code doit correspondre.
3
Tu restes connecté automatiquement à chaque visite. Utilise le bouton Déconnexion en bas à gauche pour sortir.
🎬Vidéos
1
L'onglet Vidéos affiche les vidéos publiées par tous les utilisateurs. Fais défiler vers le bas pour voir les suivantes.
2
Appuie sur une vidéo pour la mettre en pause ou relancer. Les vidéos se lancent automatiquement en scrollant.
3
Utilise les boutons à droite : 👍 Like, 👎 Dislike, 💬 Commentaires, ⬇️ Télécharger (enregistre la vidéo dans tes téléchargements).
4
Pour publier une vidéo, va dans Compte → Publications et clique sur le bouton 🎬 Vidéo.
✉️Messages & Messagerie privée
1
L'onglet Messages (avec badge rouge) affiche tous les messages reçus. Clique dessus pour répondre directement.
2
L'onglet Privé te permet d'écrire à un ami. Utilise la barre de recherche pour trouver un utilisateur et l'ajouter.
3
L'onglet Éphémère envoie des messages qui disparaissent après 24h.
👥Groupes
1
Clique sur l'onglet Groupe puis sur le bouton + pour créer un nouveau groupe.
2
Donne un nom au groupe et sélectionne les membres à inclure.
3
Tous les membres du groupe peuvent lire et envoyer des messages dans le groupe.
👤Mon Compte
1
Photo de profil : Clique sur ton avatar dans l'onglet Compte pour choisir une photo depuis ta galerie.
2
Publications : Publie du texte, des photos ou des vidéos. Les autres utilisateurs peuvent les liker.
3
Téléchargements : Les vidéos que tu télécharges depuis le fil sont sauvegardées ici. Tu peux les télécharger sur ton appareil.
4
Notifications : Vois ici toutes tes activités récentes (likes, messages, nouveaux membres...).
